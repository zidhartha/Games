## Installation

1. **Clone the Repository**:
    ```sh
    git clone https://github.com/zidhartha/snake-game.git
    ```
2. **Navigate to the Project Directory**:
    ```sh
    cd snake-game
    ```
3. **Run the Game**:
    - Make sure you have Java installed.
    - Compile the code:
      ```sh
      javac SnakeGame.java
      ```
    - Run the game:
      ```sh
      java SnakeGame
      ```
